####像开始菜单一样漂亮的菜单
```
   技术要点：MenuStrip控件中的子项ToolStripMenuItem已经包含了左侧的彩条，可以用来事项开始菜单一样的菜单，如果要该表左侧的彩条，则在对应的菜单项设置相应的图片。 
  
创建一个项目，将其命名为LikeStartMenu
从工具箱中向Form1窗体添加MenuStrip控件
为MenuStrip添加子项
为子项添加相应的图片
```
